# prettier-config

> Reusable prettier config

## Install

```
$ yarn add --dev @shelf/prettier-config
```

## Usage

In your `package.json`

```json
{
  "prettier": "@shelf/prettier-config"
}
```

## Config Content

```js
module.exports = {
  printWidth: 100,
  singleQuote: true,
  bracketSpacing: false,
  trailingComma: 'none'
};
```
